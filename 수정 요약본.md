# 📋 스토리보드 수정 전/후 비교 요약본

| 구분            | 수정 전                   | 수정 후                                 |
| --------------- | ------------------------- | --------------------------------------- |
| **패키지 핵심** | 음각(debossed) 전통 문양  | **투명 3D 입체 조각(글래스 질감)**      |
| **제형 표현**   | 가벼운 파우더가 흩날림    | **촉촉한 윤광(dewy) 제형**              |
| **피부 무드**   | 맑고 투명한 피부(glowing) | **물기 머금은 글래스 스킨(glass skin)** |

## 🔍 씬별 상세 비교

### 씬 1 (Intro)

- **수정 전 의도:** 흰 배경에 물방울이 떨어지며 퍼지는 파동으로 '청량함·깨끗함' 전달
- **문제:** 물방울/파동은 '깨끗함'이라는 추상 무드만 전달할 뿐, **제품 자체와의 연결고리가 약함**
- **수정 후 변경:** 제품의 **투명한 3D 전통 문양 조각**에 빛이 산란되는 매크로 샷으로 교체
- **결과 변화:** 추상적 이미지 → **제품의 실제 텍스처(투명함)**로 '깨끗함'을 직접 증명. 인트로부터 제품을 각인

### 씬 2 (Product Reveal)

- **수정 전 의도:** 화이트 패키지에 새겨진 전통 문양을 빛/그림자로 노출 (음각 강조)
- **문제:** '음각(debossed)'은 평면적이라 시각적 임팩트가 약하고, 차별화 포인트가 다소 밋밋함
- **수정 후 변경:** 음각 → **두껍고 투명한 글래스 질감의 입체(3D) 전통 매듭/구름 조각**으로 변경
- **결과 변화:** 평면적 음각 → **입체적·고급스러운 글래스 오브제**로 USP(차별화된 패키지) 강화

### 씬 3 (Texture & USP)

- **수정 전 의도:** 백자 위로 파우더 입자가 가볍게 흩날려 '가벼움' 표현
- **문제:** 파우더 연출은 '가벼움'엔 맞지만, **쿠션 파운데이션의 핵심인 '촉촉한 밀착력(제형)'을 전달 못함**
- **수정 후 변경:** 전통 **보자기 천 퍼프**가 스펀지를 누르고 **수분감 제형이 촉촉하게 스며드는** 연출로 변경
- **결과 변화:** 추상적 '가벼움' → **실제 제형의 촉촉함**을 보여줌. 전통 도구(천 퍼프)로 한국적 스토리도 강화

### 씬 4 (Model & Mood)

- **수정 전 의도:** 한복 소재 의상의 여배우, 맑고 투명한 피부 + 바람에 날리는 머리칼
- **문제:** 콘셉트는 좋으나 '맑은 피부'의 방향성이 다소 모호
- **수정 후 변경:** 피부 표현을 **'윤광(Glass Skin)'으로 명확히 통일** (물기 머금은 듯한 광)
- **결과 변화:** 전 씬의 '촉촉한 제형'과 **인과관계 연결** → "이 제품을 쓰면 이런 윤광 피부가 된다"는 메시지 완성

### 씬 5 (Outro)

- **수정 전 의도:** 제품 정물 고정샷 + 로고/슬로건 페이드인
- **문제:** 제품만 보여줘 마무리의 **여운·감성 임팩트가 부족**
- **수정 후 변경:** **모델(여배우)의 립싱크 컷** 추가 + 제품/텍스트 각인
- **결과 변화:** 정물 중심 → **인물 중심 마무리**로 브랜드 호감도와 슬로건 전달력 강화

## 💡 종합 인사이트

**일관된 수정 방향: "추상적 무드 → 구체적 제품 증명"**

1. **추상 → 구체:** 물방울·파우더 같은 은유적 표현을 줄이고, **제품의 실제 텍스처·제형·패키지**를 직접 보여주는 방향으로 전환
2. **하나의 키워드로 통일:** 전체를 **'투명함(transparent/glass)'** 키워드로 묶어 인트로~아웃트로의 메시지 일관성 확보
3. **인과관계 강화:** "촉촉한 제형(씬3) → 글래스 스킨(씬4)"으로 제품 효과의 **논리적 연결** 구축
4. **효율적 길이:** 30초 → 20초로 압축하여 **핵심 메시지에 집중**
